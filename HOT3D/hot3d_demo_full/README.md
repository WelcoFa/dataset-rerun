# HOT3D-Clips

This Hugging Face repository hosts [HOT3D-Clips](https://github.com/facebookresearch/hot3d/tree/main/hot3d/clips), a set of curated sub-sequences of the [HOT3D dataset](https://facebookresearch.github.io/hot3d/).

Download instructions for HOT3D-Clips and the full HOT3D dataset can be found [here](https://facebookresearch.github.io/hot3d/#download).

See [HOT3D Toolkit](https://github.com/facebookresearch/hot3d/tree/main/hot3d/clips) for documentation of the data format and for Python utilities (for loading, undistorting fisheye images, rendering using fisheye cameras, etc.).

More details can be found in the [HOT3D paper](https://arxiv.org/pdf/2411.19167) and [BOP 2024 report](https://arxiv.org/pdf/2504.02812).
